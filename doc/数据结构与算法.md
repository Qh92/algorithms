# 数据结构与算法

## 1.异或运算

两值相等异或为0，两值不等异或为1。异或运算可以理解为**无进位相加**

```java
int a = 10110;
int b = 00111;
a ^ b = 10001;
```

#### 特性：

1）0 ^ N = N   N ^ N = 0

2）异或运算满足，交换律、结合律

a ^ b = b ^ a; (a ^ b) ^ c = a ^ (b ^ c);

3）有多个数异或成为N，这些数谁先异或谁后异或无关，结果都一样。

eg.

```java
//注意：能这么玩的前提是 a,b在内存中是两块独立的区域
int a = 10;
int b = 33;
a = a ^ b;
b = a ^ b;// b = a ^ b ^ b = a ^ (b ^ b) = a ^ 0 = a;
a = a ^ b;// a = a ^ b ^ a = b ^ (a ^ a) = b;
//最终a、b数值交换
```
