```java
private static void bubbleSort(int[] arr){
    //临时变量
    int temp;
    //标识变量，表示是否进行过交换
    boolean flag = false;
    for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j+1]){
                flag = true;
                temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;

        }

            flag = false;
        }

}
```

```java
public static void insertionSort(int[] arr) {
    if(arr == null || arr.length < 2) {
        return;
    }
    // 0~0有序，0~i想有序
    for(int i = 1; i < arr.length; i++) {
        for(int j = i - 1; j >= 0 && arr[j] > arr[j + 1]; j--) {
            swap(arr,j,j+1);
        }
    }
}
```

#### hard***:局部最小值问题

```markdown
在一个无序arr中，相邻两个数一定不相等，求局部最小值（只求一个局部最小值），时间复杂度好于O(N)
解析：如果 0~1范围，0位置的数比1位置的数小，则0位置的数是局部最小值。如果 n-2~n-1范围，n-1位置的数比n-2位置的数小，则n-1位置的数是局部最小值。如果i位置的数比i-1位置的数小，比i+1位置的数小，则i位置的数是局部最小值。
1.先在0~n-1范围中通过二分，找到中间值i，如果中间值i比i-1和i+1的数都小，i即为局部最小值，则可以直接返回中间值i。
2.如果i比i-1的值大，则可以在0~i-1范围继续二分，依次类推，二分完整个数据。
```

### 8.异或运算

两值相等异或为0，两值不等异或为1。异或运算可以理解为**无进位相加**

```java
int a = 10110;
int b = 00111;
a ^ b = 10001;
```

#### 特性：

1）0 ^ N = N   N ^ N = 0

2）异或运算满足交换律、结合律

a ^ b = b ^ a; (a ^ b) ^ c = a ^ (b ^ c);

3）有多个数异或成为N，这些数谁先异或谁后异或无关，结果都一样。

eg.

```java

```

### 1.master公式

$$
T(N) = a * T(N/b) + O(N^d) \\
T(N):母问题的规模 \\
T(N/b):子问题的等规模 \\
a:子问题被等规模调用多少次 \\
O(N^d):除了子问题的调用之外其它的时间复杂度是多少
$$

![image-20211230230335795](assets\image-20211230230335795.png)



```java
public static int getMax(int[] arr) {
    return process(arr,0,arr.length - 1);
}

/**
  * arr[L..R]范围上求最大值
  * @param arr
  * @param L
  * @param R
  * @return
  */
private static int process(int[] arr, int L, int R) {
    // 数组上只有一个数
    if (L == R) {
        return arr[L];
    }
    // 中点，防止 (L+R)/2 (L+R)溢出
    int mid = L + ((R - L) >> 1);
    int leftMax = process(arr, L, mid);
    int rightMax = process(arr, mid + 1, R);
    return Math.max(leftMax, rightMax);
}
```

```markdown
如上，process()子问题规模，每次调用两次，即a = 2，每次调用的规模都是1/2，即b = 2，剩余的时间复杂度为O(1)，即d = 0;满足master公式：T(N) = 2 * T(N/2) + O(1)
时间复杂度：log(b,a) == 1 > d == 0 -> O(N) 等效于从左到右遍历一次求最大值
```





### 2.归并排序

![image-20211230230948417](assets\image-20211230230948417.png)

```java
public static void process(int[] arr,int L,int R) {
    if (L == R) {
        return;
    }
    int mid = L + ((R - L) >> 1);
    process(arr,L, mid);
    process(arr, mid + 1, R);
    merge(arr,L,mid,R);
}

private static void merge(int[] arr, int L, int mid, int R) {
    int[] help = new int[R - L + 1];
    int i = 0;
    int p1 = L;
    int p2 = mid + 1;
    while (p1 <= mid && p2 <= R) {
        help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];
    }
    while (p1 <= mid) {
        help[i++] = arr[p1++];
    }
    while (p2 <= R) {
        help[i++] = arr[p2++];
    }
    for (int j = 0; j < help.length; j++) {
        arr[L + j] = help[j];
    }
}
```

小和问题分析：

```markdown
可以将小和问题换个方向思考：当前这个数，找到右边比它大的数，如果能找到，当前这个数就是一个小和
```

```java
public static int process(int[] arr,int l,int r) {
    if (l == r) {
        return 0;
    }
    int mid = l + ((r - l) >> 1);
    return  process(arr, l, mid) + process(arr, mid + 1, r) + merge(arr, l, mid, r);
}

private static int merge(int[] arr, int l, int mid, int r) {
    int i = 0;
    int[] help = new int[r - l + 1];
    int p1 = l;
    int p2 = mid + 1;
    int res = 0;

    while (p1 <= mid && p2 <= r) {
        // 小和增加的数量
        res += arr[p1] < arr[p2] ? (r - p2 + 1) * arr[p1] : 0;
        help[i++] = arr[p1] < arr[p2] ? arr[p1++] : arr[p2++];
    }
    while (p1 <= mid) {
        help[i++] = arr[p1++];
    }
    while (p2 <= r) {
        help[i++] = arr[p2++];
    }
    if (help.length >= 0) {
        System.arraycopy(help, 0, arr, l, help.length);
    }
    return res;
}
```

```java
        help[i++] = arr[p1++];
```

```java
public static void quickSort(int[] arr,int left,int right){
    int middle;
    if(left < right){
        middle = partition(arr,left,right);
        quickSort(arr,left,middle-1);
        quickSort(arr,middle+1,right);
    }
}

public static int partition(int[] arr,int left,int right){
    int pivot = arr[left];
    while(left < right){
        while(left < right && arr[right] >= pivot) {
            right--;
        }
        arr[left] = arr[right];
        while(left < right && arr[left] <= pivot) {
            left++;
        }
        arr[right] = arr[left];
    }
    arr[left] = pivot;
    return left;
}
```

```java
public static void heapInsert(int[] arr,int index) {
    while(arr[index] > arr[(index - 1) / 2]) {
        swap(arr,index,(index - 1) / 2);
        index = (index - 1) / 2;
    }
}
```

##### heapify:从上往下调整

现在有这么一个需求，用户给定一些数，这些数依次调成了大顶堆。此时heapSize = 6。

用户想要知道此时堆中的最大值是多少，并将这个数在堆中去掉。

![image-20220108223844019](assets\image-20220108223844019.png)

将堆顶值用变量保存起来，将最后的值4移到堆顶，此时让4和它的左右两个结点进行比较，大值放到堆顶，如果位置有交换，则依次往下比较值，直到不能比较为止。



```java
// 某个数在index位置，能否往下移动
public static void heapify(int[] arr,int index,int heapSize) {
    // 左孩子的下标
    int left = index * 2 + 1;
    // 下方还有孩子的时候
    while(left < heapSize) {
        // 两个孩子中，谁的值大，把下标给largest
        int largest = left + 1 < heapSize && arr[left + 1] > arr[left] ? left + 1 : left;
        // 父和孩子之间，谁的值大，把下标给largest
        largest = arr[largest] > arr[index] ? largest : index;
        if(largest == index) {
            break;
        }
        swap(arr,largest,index);
        index = largest;
        left = index * 2 + 1;
    }
}
```

| 插入排序     | O(N^2)     | O(1)       | 稳定   |
| 归并排序     | O(N*logN)  | O(N)       | 稳定   |
| 快排（随机） | O(N*logN)  | O(logN)    | 不稳定 |
| 堆排序       | O(N*logN)  | O(1)       | 不稳定 |

**一般来讲，排序优先选择快速排序，经过实验，其常数时间最快的**
